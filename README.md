# Zadatak 18:

Koristeći Rx.NET i SportMonks API, prikupiti informacije o fudbalskim utakmicama koristeći opciju pretrage po postavama (fixtures). Rx vrši osnovno mapiranje dobijenih podataka i emituje ih kao poruke aktorima, uključujući i polje sa informacijama o igračima (lineups). Aktori čuvaju podatke o igračima po utakmici kao interno stanje i evidentiraju ime, prezime, godinu rođenja, broj dresa i zemlju svakog igrača. Web server prima zahtev sa identifikatorom utakmice i prevodi ga u poruke aktorima. Prikazati dobijene rezultate.

Dokumentacija dostupna na linku: https://docs.sportmonks.com/football
